Upload File to Google Drive

Localhost Setup: To upload files from your computer to Google Drive, you must:
1. Serve this page with a local web server (not file://).
2. Place secret.json in the same folder as this HTML.
3. Add http://localhost and your port (e.g. http://localhost:8000) to the OAuth client in Google Cloud Console.
4. Use the "Connect Google Drive" button, then "Upload Files".

Example: python -m http.server 8000 then open http://localhost:8000/simple-upload.html
